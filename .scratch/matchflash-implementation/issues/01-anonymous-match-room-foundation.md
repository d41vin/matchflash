# 01 — Anonymous Match Room foundation

**What to build:** A mobile-first Lobby and global Match Room that anonymous fans can open directly, backed by a fixture-scoped Match Room projection.

**Blocked by:** None — can start immediately.

**Status:** ready-for-agent

- [ ] Anonymous visitors can browse fixtures and open a global Match Room without an account or wallet.
- [ ] The Match Room renders a coherent fixture state from one safe projection and never exposes raw provider payloads.
- [ ] The initial experience is responsive and covered by behavior-focused tests.
